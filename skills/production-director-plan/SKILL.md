# 导演规划

## 输入
分集剧本文本（Markdown 格式）。

## 输出格式
Markdown，每场戏一个段落：

### 场景1 | 内·咖啡馆·黄昏

**调度**：人物 A 坐窗边 → B 推门入 → A 抬头

**镜头**：从 A 的过肩看 B → 反打 B 中景 → 双人全景

**视觉**：暖色黄昏光，浅景深，手持轻微晃动

## 规划原则
1. 每场标注调度（blocking）、镜头方案、视觉参考
2. 对话场景用 180 度原则布局
3. 动作场景标注关键姿态（起→高潮→收）
4. 给出整体视觉风格参考（色板/光影关键词）
