# 分集剧本

## 输入
- 改编策略（AdaptationStrategy）
- 章节原文

## 输出格式
Markdown，每集以 `## E01` 开头：

```
## E01 | 集标题

场景：内/外 · 地点 · 时间

动作描写（自然段，不含镜头语言）。

角色名：（状态/表情）台词内容
```

## 格式规则
- `## E编号 | 集标题` 为集头
- `场景：` 为场景头，格式固定
- 动作描写用自然段
- 对白格式：`角色名：（表情）台词`
- 每集 5-10 场，每场 30-60 秒

## 原则
1. 不写镜头语言（景别、角度、运镜）
2. 心理描写转为表情/动作或画外音
3. 角色语言风格保持一致
4. 每集结尾留悬念/钩子
