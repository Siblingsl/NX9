# 分镜表

## 输入
导演规划或剧本文本。

## 输出格式
JSON 数组，每行一个镜头：

```json
{
  "id": "唯一ID",
  "group": "S01",
  "shotSize": "CU",
  "cameraMove": "推",
  "durationSec": 4,
  "descriptionZh": "画面描述",
  "dialogue": "对白",
  "sfx": "音效",
  "videoDesc": "视频动态描述",
  "associateAssetIds": []
}
```

## 分镜规则
1. **一个镜头一个动作**：不把多个动作塞进一镜
2. **首镜定调**：第一镜确立空间、氛围、主体
3. **情绪转折分镜**：每个情绪转折点断一次
4. **运镜可执行**：只写能生成的运镜（推、拉、摇、移、跟、固定）
5. **连续性**：相邻镜头的主体位置、光线、服装保持一致
6. **时长合理**：对话 3-4s，动作 4-6s，环境 4-6s
7. **S-Class 分组**：单组 ≤15s，场景变化处断开
