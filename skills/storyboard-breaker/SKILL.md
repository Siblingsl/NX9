# 分镜拆解

## 输入
一段故事或剧本文本。

## 输出格式
Markdown 表格，每个镜头一行，控制每镜 3-6 秒：

| 镜号 | 景别 | 画面描述 | 英文提示词 | 时长 | 备注 |

## 拆解原则
1. **一个镜头一个动作**：不把多个动作塞进一镜
2. **首镜定调**：第一镜确立空间、氛围、主体
3. **情绪转折分镜**：每个情绪转折点断一次
4. **运镜可执行**：只写 AI 视频/图像能生成的运镜（推、拉、摇、移、跟、静止）
5. **连续性**：相邻镜头的主体位置、光线、服装保持一致
6. **时长合理**：人物对话镜 3-4s，环境交待镜 4-6s

## 提示词要求
英文提示词列必须自包含（self-contained），可直接喂给生成模型，包含主体 + 场景 + 光线 + 风格。
