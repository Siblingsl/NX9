# 角色配音分配

## 任务
为剧本中每个角色推荐一组配音参数：**音色（voice）+ 语速（speed）+ 情绪（emotion）**。

## OpenAI 兼容音色池
alloy · echo · fable · onyx · nova · shimmer
（若使用其他 TTS 提供方，请改用对应音色 ID）

## 判断维度
1. **性别年龄**：对齐音色性别与年龄段
2. **性格**：沉稳/活泼/阴郁/温柔 → 对应低沉/明亮/沙哑/柔和音色
3. **场景情绪**：紧张戏加快语速，舒缓戏减慢

## 输出格式
Markdown 表格：

| 角色 | voice | speed | emotion | 理由 |

## 原则
- 一部戏内同一角色始终用同一音色（声音一致性）
- 旁白/画外音与角色音色区分开
- speed 范围 0.75–1.25，默认 1.0
- 给出简短理由，便于人工微调
