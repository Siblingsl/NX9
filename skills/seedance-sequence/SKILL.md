# 连续剧情模板

```text
项目目标：[故事最终要到达的结果]
已发生：[已经被接受的视频事实]
本段只拍：[本 clip 的一个可见任务]
不能提前出现：[保留到后面 clip 的内容]
参考：Image1 锁定主体；Video1 仅参考运镜；Audio1 仅参考节奏
提示词：[一个动作 + 一个镜头 + 真实光线 + 声音]
```

## 原则
- Clip 01 完成并确认后，再写 Clip 02
- 每段只拍一个可见任务
- 输出时标注 clip 序号
