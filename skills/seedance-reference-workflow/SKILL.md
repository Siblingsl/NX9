# 多模态引用工作流

| 类型 | 用途 | 写法 |
|------|------|------|
| Image | 角色/产品外观锁定 | Image1 为角色参考，严格保持… |
| Video | 仅参考运镜节奏 | Video1 仅参考运镜，不改变主体 |
| Audio | 节奏/环境声 | Audio1 仅参考节奏 |

## 上限提醒（Seedance 2.0）
- 图片参考 ≤9
- 视频参考 ≤3
- 音频参考 ≤3
- prompt ≤5000 字符
