# 剧本改写指南

## 改写原则
1. **保留核心情节**：不改变主线和角色关系
2. **增强画面感**：叙述转为可视化场景描写
3. **对话驱动**：用对白推动情节，减少旁白
4. **节奏把控**：每场戏 30-60 秒，适合短视频
5. **不写镜头语言**：景别、角度、运镜交给分镜步骤

## 格式
```
## S01 | 内景 · 咖啡厅 | 黄昏

动作描写（自然段，不含镜头语言）。

角色名：（状态/表情）台词内容
```

## 格式规则
- `## S编号 | 内景/外景 · 地点 | 时间段` 为场景头
- 动作描写用自然段
- 对白格式：`角色名：（表情）台词`

## 注意
- 心理描写转为表情/动作或画外音
- 长叙述拆成多场短戏
- 角色语言风格保持一致
- 场景编号连续递增（S01, S02…）
- 时间段要具体（黄昏、深夜、清晨），不写笼统的"白天"
