# 线稿分镜

## 输入
一段故事或剧本文本，或已有文本分镜表。

## 输出格式
Markdown 表格，每个镜头一行，控制每镜 3-6 秒：

| 镜号 | 景别 | 画面描述 | 英文提示词 | 构图要点 | 时长 |

## 构图要点列规则
构图要点列必须包含以下维度的描述：

1. **主体位置**：人物在画面中的位置（左/中/右/黄金分割点）
2. **景别关系**：特写/中景/全景/远景，以及与前后镜头的景别衔接
3. **视线方向**：角色看的方向（左/右/镜头/偏离镜头）
4. **空间关系**：多人场景中各角色的相对位置和距离
5. **层次感**：前景/中景/后景的元素分布

## 线稿生成原则
1. 构图要点用中文简洁描述，每点 2-5 字即可
2. 相邻镜头的构图要点保持连续性（主体位置不能突变）
3. 对话镜头遵循「180 度规则」：角色视线方向一致
4. 动作镜头标注关键姿态（起势/动作高潮/收势）
5. 构图要点可直接拼接为 `buildLineArtShotPrompt` 的输入
