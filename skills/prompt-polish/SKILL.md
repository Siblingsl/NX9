# 提示词润色

## 目标
把用户随口写的提示词，润色成结构清晰、要素齐全、可直接用于图像/视频生成的提示词。

## 五段结构
优秀提示词按以下顺序组织：

1. **主体**（Subject）：谁 / 什么，核心动作
2. **环境**（Environment）：场景、地点、背景
3. **光线**（Lighting）：时间、光质、光位
4. **风格**（Style）：艺术风格、镜头、质感
5. **参数**（Quality）：分辨率、细节强度、负面词

## 润色原则
- **去冗余**：删除"很漂亮、很好看"等无效词，换成具体描述
- **补缺失**：缺哪段补哪段，但不臆造用户没说的主体
- **英文化**：最终输出英文，逗号分隔
- **保留意图**：用户强调的元素必须保留并强化

## 输出
直接输出润色后的英文提示词，不解释。
负面词用 `--no` 或单独的 negative prompt 行给出（视下游模型而定）。
